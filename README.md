# ThinkEvernote
evernote sync to blog website juest useing evernote apps.

使用印象笔记应用同步更新网站博客文章.

# Application Path tree
 |-application //应用目录
 |-composers // evernote sdk & ThinkPHP 5.0 RC3 类库
 |-extend // 第三方扩展目录
 |-runtime // 运行时文件
 |-src // 对外目录
 |---assets // 静态资源文件目录
 |---data // 配置文件目录
 |-views // 前端模板目录
 |---default // 网站默认模板
 |---administrator // 管理后台模板
 |---installer // 安装界面模板
